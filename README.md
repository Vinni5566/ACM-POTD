# ACM-POTD Repository

This repository contains my daily solutions for the ACM POTD (Problem of the Day) Event organized by ACM IGDTUW.

---

## 📌 Event Details

* Duration: 22 March 2026 – 25 April 2026
* Total Days: 34
* Requirement: Solve 1 question per day
* Certificate Criteria: Minimum 30 problems solved

---

## 🔗 Important Links

* POTD Website: https://acmigdtuw-potd.vercel.app/

---

## 📂 Repository Structure

Each day contains a Markdown file in the format:

Fullname_Date_POTD-day.md

Example:
Pixiu_22-03-26_Day-1.md

Each file includes:

* Problem description
* Approach
* Solution
* Screenshot of accepted submission
